# 🛡️ SafeVoice AI: Real-Time Audio Forensics

A deep-learning-powered dashboard designed for banks to detect AI-cloned voices in real-time call center streams. This project analyzes audio spectrograms to identify "synthetic scars" left by Generative AI.

## 🚀 Quick Start (For Contributors)

### 1. Clone the Repository
```bash
git clone https://github.com
cd AudioForensics


### 2. Frontend Setup (React + Vite)
cd frontend
npm install
npm run dev

The dashboard will be available at http://localhost:5173


### 3. Backend Setup (FastAPI + AI Engine)
# Open a new terminal
cd backend
pip install -r requirements.txt
python main.py
